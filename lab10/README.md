Лабораторна робота №10  
Тема: Вступ до покажчиків  
Завдання №1
Дінара Баршакаєва
Рік: 2025

Опис завдання:
Реалізувати програму, яка:
- отримує масив з N цілих чисел;
- знаходить усі пари елементів, що розташовані між першим та другим від’ємними елементами початкового масиву;
- рахує кількість таких пар;
- виводить отриманий результат.

Вимоги:
- використовувати динамічне виділення пам’яті;
- доступ до елементів масиву виконувати через розіменування покажчиків;
- заборонено використовувати глобальні змінні;
- заборонено використовувати функції введення/виведення всередині бібліотеки;
- компіляція та запуск через Makefile;
- структура проєкту відповідає вимогам (bin, src, doc, test);
- необхідно забезпечити відсутність витоків пам’яті (перевірка через valgrind);
- код має бути розбитий на модулі (main.c, lib.c, lib.h).

Структура директорій:
lab10/
  bin/
  doc/
    lab10.txt
  src/
    main.c
    lib.c
    lib.h
    Makefile
  test/

Короткий опис модулів:
- main.c: вводить дані, викликає функції бібліотеки, виводить результат.
- lib.h: містить прототипи функцій.
- lib.c: реалізація логіки обчислення результату.
- Makefile: збірка програми (створення bin/app).

Запуск:
1. make
2. ./bin/app

2. ./bin/app
