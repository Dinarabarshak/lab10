# Лабораторна робота №10

**Тема:** Вступ до покажчиків  
**Автор:** Дінара Баршакаєва (2025)

## Опис завдання
- Задано масив з `N` цілих чисел.
- Необхідно визначити мінімальний та максимальний елементи.
- Обчислити суму елементів, розташованих між цими позиціями (у порядку появи в початковому масиві).
- Сформувати новий динамічний масив, що містить елементи між мінімумом та максимумом.
- Доступ до елементів здійснюється через розіменування покажчиків без використання індексування `[]`.
- Масиви мають виділятися динамічно; використання функцій введення/виведення у коді заборонено.

## Структура проекту
```
lab10/
├── Makefile           # збірка бінарних файлів, тести, valgrind
├── README.md          # опис завдання та інструкції
├── src/
│   ├── lib.c          # реалізація логіки побудови масиву між мін/макс
│   ├── lib.h          # прототипи та документаційні коментарі
│   └── main.c         # приклад використання бібліотеки з константними даними
├── test/
│   └── test.c         # модульні тести на основі assert
└── doc/
    └── lab10.txt      # файл для звіту/doxygen (порожній шаблон)
```

## Команди
- **Збірка основної програми та тестів:** `make`
- **Запуск модульних тестів:** `make test`
- **Перевірка витоків пам'яті (valgrind):** `make leak-check`
- **Очищення артефактів збірки:** `make clean`

Усі зібрані файли розміщуються у каталозі `dist/`.
